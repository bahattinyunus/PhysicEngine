## Güvenlik Zafiyetlerini Bildirme

PhysicEngine'de bir güvenlik açığı bulduğunuzda lütfen doğrudan GitHub üzerinden bir Issue açmak yerine, gizlilik çerçevesinde aşağıdaki adımları izleyin:

1. Zafiyeti [bahattinyunuscetin@gmail.com](mailto:bahattinyunuscetin@gmail.com) adresine bildirin.
2. Size 48 saat içinde geri dönüş yapılacaktır.
3. Çözüm uygulanana kadar lütfen zafiyeti halka açık şekilde paylaşmayın.

Güvenliğe verdiğiniz önem için teşekkür ederiz!
